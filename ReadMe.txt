Verserve

This is a poetry database project, meaning that the basic content takes the form
of text strings which are grouped in sections which in turn are grouped in
poems.

Modules added
* sqlite-jdbc

Eclipse tips
* After creating a dynamic web project, downgrade it to a v5.0 dynamic web module (in project facets).
* After converting to a maven project, it might be necessary to downgrade the project to a v5.0 dynamic web module.
